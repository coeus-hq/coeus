    Students can:

    • View their courses
    • Search for instructor made courses
    • Enroll in instructor made courses
    • Join & participate in live sessions
    • Be promoted to teacher assistants by instructors for a specific course section
    • Set their timezone

    Instructors can:

    • View their courses on home page
    • Create courses, read courses, update courses, delete courses
    • Start & end live sessions
    • Prompt students to teacher assistants for a specific course section
    • View automated attendance after a live session has ended

    The admin can:

    • View all users
    • Create users, read users, update users, delete users
    • Create or promote a user to instructor
    • Change organization settings (name, logo, timezone, etc.)
    • Reset the database and trigger an onboarding wizard
    • Seed the database with default seed data
